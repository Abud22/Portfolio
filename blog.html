U htmlu imamo dvije vrste elemenata:

  1. Block elementi
  2. Inline elemnti

Elementi na nivou bloka

Element na nivou bloka uvijek započinju na novoj liniji i zauzimaju punu dostupnu širinu (proteže se lijevo i desno koliko god može).

Inline Elementi

Inline elementi ne započinju u novoj liniji i zauzimaju samo onoliko širine koliko je potrebno.

Imamo tri nacina da implementacije style sheet:

  • Eksterni CSS
  • Interni CSS
  • Inline CSS

Eksterni CSS

Pomoću vanjskog (eksterne )lista stilova možete promijeniti izgled cijele web stranice tako što ćete promijeniti samo jednu datoteku!

Interni CSS

Interna css se može koristiti ako jedna HTML stranica ima jedinstveni stil. Interni stil definiran je unutar elementa style, unutar odjeljka glave.

Inline CSS

Ugrađeni stil ili inline css, može se koristiti za primjenu jedinstvenog stila na jedan element. Da biste koristili umetnute stilove, dodajte atribut style odgovarajućem elementu. Atribut style može sadržavati bilo koje CSS svojstvo.
